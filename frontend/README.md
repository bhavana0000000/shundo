# Shundo Frontend

## Setup
npm install

## Run (backend must be running on port 8001)
npm run dev

Then open http://localhost:5173

## Pages
- /          landing page
- /login     real Google OAuth sign-in (or guest mode)
- /app       dashboard - give it a goal, watch it think live over WebSocket
- /app/budget  real budget tracker, connected to backend SQLite data
